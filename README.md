# Hospital_Management_System
PFE Final Semester Project 2026

A menu-driven Hospital Management System built in C++ as a final project for Programming Fundamentals (CS1011) - Spring 2026, FAST-NUCES Islamabad.

## Features
- Patient Management (Add, Update, Delete, Search)
- Doctor Management (Add, Update, Delete, Search)
- Appointment Scheduling with balance check
- Treatment & Billing records
- File-based persistent storage
- Data cleaning and validation on startup
